SVどうぐプリンター出力制御自動化 ver.0.0.3 作成者：フウ(@dragonite303)、ジュナリ(@junari000)

■要約
　本プログラムはSVの道具プリンターについて何度も同じ出力を繰り返すよう制御するプログラムです。
　Poke-Controller MODIFIED, Poke-Controller MODIFIED Extensionのどちらかで使用可能です。

■開発環境
　python v3.7.9

■事前準備
* (Extension版の場合)Item_Printer_ver_0_0_3のフォルダをそのままPythonCommandsにコピーしてください。
* (Modified版の場合)Item_Printer_ver_0_0_3のフォルダをそのままPythonCommandsにコピーしてください。
　その後、Item_Printer_ver_0_0_3の中にあるItem_PrinterのフォルダをTemplateフォルダに移動させてください。

■使用方法
* 道具プリンターの前に立って話しかけられる状態にしてプログラムを実行します
* プリセットを選択します。なければそのまま「OK」を押してください。デフォルト値を使用します。
* 繰り返し回数:日時1と日時2を何回ループするか入力してください
* 日時1/待機時間1/プリント回数1:チャンスにするための日時および待機時間、出力するアイテムの数を設定します
* 日時2を使う:ほしいアイテムを出力する際に日時変更を行うかを設定します
* 日時2/待機時間2/プリント回数2:ほしいアイテム(オシャボなど)を排出するための日時および待機時間、出力するアイテムの数を設定します。プリント回数2で1や5を選んだ場合はチャンスが消えるまでは日時1をスキップし、日時2で日時変更を行います。（1の場合は10回、5の場合は2回）
* おまかせ選択:基本的にチェックを入れてください(外した場合は今後対応予定)
* DEBUG:排出時の画面をキャプチャします
